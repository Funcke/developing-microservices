# Requirements

- Google Cloud Platform Account
- git
- gcloud
- kubectl

<br>
<br>

### Step 1: Clone repository:

Execute in terminal:

```
git clone https://github.com/dynatrace-innovationlab/developing-microservices.git
```

<br>

### Step 2: Prepare gcloud

Execute in terminal:

```
gcloud auth
```

Execute in terminal:

```
gcloud init
```
